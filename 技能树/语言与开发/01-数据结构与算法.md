找了很多资料，总结出来一下几种路线：

路线1：

```
数据结构与算法的基础思维 -> 动态规划思维 -> 数据结构思维 -> 算法思维
```

路线2：

```
数组 -> 链表 -> 哈希表 -> 字符串 -> 栈与队列 -> 树 -> 回溯 -> 贪心 -> 动态规划 -> 图论 -> 高级数据结构
```

我觉得这两个路线都没毛病，但是适不适合我呢？我准备两个结合起来实践一下试试。
